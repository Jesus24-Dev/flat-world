Trapecio Isósceles

Los dos lados no paralelos tienen la misma longitud. Además, los ángulos que se forman entre cada base y un lado no paralelo son iguales.

Trapecio escaleno

Ninguno de sus lados ni ángulos es igual. Es decir, todos sus lados tienen longitudes diferentes y todos sus ángulos tienen medidas diferentes.

Cálculo de Área de un trapecio

A = ((B + b ) * h ) / 2

Para hallar el área de un trapecio, debes conocer las longitudes de los dos lados paralelos (las "bases") y la altura. Suma las longitudes de las dos bases y luego multiplica por la altura. Por último, divide entre 2 para obtener el área del trapecio
